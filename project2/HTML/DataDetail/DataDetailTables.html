Data Detail Tables


DetailConfig

This table contains the configuration for a Dynamic Detail dialog.

The Primary key is ID.

The Unique key is UserID and Name.

The data key is DataConfigName and TableName. The DataConfigName references the database configuration from the file DataConfigs.xml.

The PageColumnsLimit identifies the maximum number of ControlColumns on each tab page.

The ColumnRowsLimit identifies the maximum number of ControlRows in each ControlColumn.

ControlColumn

This table contains the ControlColumns for each tab.

The Primary key is ID.

The parent Foreign key is DetailConfigID.

The Unique key is DetailConfigID, TabPageIndex and ColumnIndex.

ControlRow

This table contains the ControlRows for each ControlColumn.

The Primary key is ID.

The parent Foreign key is ControlColumnID.

The Unique key is ControlColumnID and RowIndex
